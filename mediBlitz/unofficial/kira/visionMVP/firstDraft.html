MediBlitz

A smarter way to study without drowning in random PDFs.

Problems

  • No proper place for study materials
  • No proper place to find practice questions

How MediBlitz Works

Everything is stored as small knowledge blocks.

What is a Block?

  • Topic Title
  • Topic Explanation
  • Blueprint for practice questions (AI will generate questions)
  • Linked prerequisite blocks

Creating a Syllabus

Manual Method

  • User searches a topic
  • Sees prerequisites and can add them
  • Preview bar shows selected topics
  • User can finalize, edit, or remove topics

AI Method

  • User uploads their syllabus
  • AI lists all required blocks
  • AI lists all prerequisites
  • User can add blocks, prerequisites, and questions

Each AI-added block costs 1 in-app currency. Currency resets daily. Users can upgrade with real money.

Practicing Questions

  • User opens syllabus
  • Goes to questions
  • Can practice quizzes
  • Later, users will upload answers and AI will evaluate them

MVP Scope

  • Blocks
  • Syllabus creation
  • Basic quizzes

Future Features

  • Leaderboards
  • More AI tools
  • Smarter practice options

Summary

MediBlitz organizes learning using knowledge blocks, auto-generates questions, and helps students build custom syllabuses manually or with AI, while tracking cost using a simple currency system.
