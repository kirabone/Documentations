MadeBy Idea-Pitch

A platform that acts as a marketplace for projects and a networking platform for the developers who built them.

The Problems We Have Discovered

  • There is no perfect platform to display projects in a way similar to Instagram posts.
  • There is no dedicated platform focused on collaborating with other serious developers.
  • There is no single platform to explore a large number of projects for inspiration and reference.

How Do We Solve This Problem?

We will build a platform named MadeBy, where developers can upload their projects as posts.

Core Features

  • Posts — Each post represents a project and includes:
    • Project link (mandatory)
    • Project preview images (optional)
    • Project preview video (optional)
    • Project preview text (optional)
    • Title (mandatory)
    • Description (optional)
    • Comment section (optional)
    • Like / Dislike section (optional)
    • Get Code button (optional — requires the developer to upload the code)
    • Contributions (mandatory, at least for self) — allows adding other developers and describing their contributions to the project

      • NOTE: The uploader of the project is not considered the owner of the project.

  • Developer Account — Each user account includes:
    • Skills list (skills are marked as "not verified" by default and can be verified using built-in tests or by uploading supporting certificates for external verification)
    • Other platform links (GitHub, YouTube, portfolio, etc.)
    • Follow button to follow developers
    • Post list containing all projects, categorized as "solo" or "collab – collab name"
    • Bio section
  • Collab
    • A group chat feature where developers can invite others to discuss project ideas and plans
    • Invites appear in a separate collab section showing sent and received requests
    • This feature functions mainly as a focused group chat
  • Inbox
    • A general chat feature allowing developers to message each other directly

Core Philosophy

Our core philosophy is to create a platform where developers can upload their projects, explore projects built by other developers, learn from them, and gain inspiration and references. Since the platform is niche-focused only on developers, every user on the platform is a developer, which increases the chances of finding suitable coding or development partners.
